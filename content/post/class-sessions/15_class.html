In-Class Exercise: Data Viz

DATA VISUALIZATION - HOW TO MAKE UGLY PLOTS

  1. First, you’re going to need to download some datasets. Here are three we will be working with in class:

  • cereal.csv - data on the nutritional content of different breakfast cereals.

  • bites.csv. Which dogs bite the most?

  • bigfoot.csv. Number of bigfoot sightings per year in North America

  1. We’re going to make a bar plot, then a scatter plot. First you read in the data using read_csv():

    • bites<-read_csv("./-------")
    • bigfoot<-read_csv("./-------")

  2. Then we will walk through this code (you can download it later): viz_code.class.R

  3. After we lear to make the basic figures, your job is to make a figure as ugly as possible. If you need some inspiration, check out these examples!
